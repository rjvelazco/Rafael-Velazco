Hopitales - Administrador

Proyecto robusto realizado con Angular intengrando un template en HTML5/CSS3 en Angular creando componentes reutilizabes. La estructura del proyecto se basa en módulo para ser facilmente escalable, implementando el patrón de diseño Lazyload para mejorar el performance del aplicativo. El proyecto incluye un login y sistema de registro que implemente Reactive Forms, Guards para validar la autenticación del usuario, cargada de archivo, y un guard especialido para permitir el acceso a los administradores.
